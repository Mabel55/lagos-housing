A Machine Learning Web Application that predicts house rent prices in Lagos, Nigeria, based on real-time market data.

## About the Project
Finding a house in Lagos is difficult, and prices vary wildly between areas like **Ikeja GRA** (Luxury) and **Ikorodu** (Affordable). 
This project uses **Artificial Intelligence** to help users estimate the fair market price for a property before they pay.

**Key Features:**
* **Predicts Rent:** Calculates price based on Location, Number of Bedrooms, POP Ceiling, and Road Accessibility.
* **Smart Distinctions:** Understands the price gap between "Luxury" and "Basic" areas.
* **Web Interface:** Simple, mobile-responsive UI built with Flask.

##  Tech Stack
* **Language:** Python 3
* **Backend:** Flask (Web Framework)
* **Machine Learning:** Scikit-Learn (Decision Tree Regressor)
* **Data Handling:** Pandas & NumPy
* **Frontend:** HTML5, CSS3
